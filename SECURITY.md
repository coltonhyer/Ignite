# Security Policy

## Supported Versions

Currently, the `main` branch is the only supported version for security updates. 

## Reporting a Vulnerability

Because Ignite is a security-focused application involving encryption and strict atomic invariants, please **do not** report security vulnerabilities by creating a public GitHub issue.

If you believe you have found a security vulnerability in Ignite, please report it via the **GitHub Security Advisory** feature ("Private vulnerability reporting" if enabled) or by contacting the repository maintainer directly. 

We take all security considerations very seriously and will triage and address the issue as quickly as possible.
